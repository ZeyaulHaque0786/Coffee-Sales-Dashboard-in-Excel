# Coffee-Sales-Dashboard-in-Excel
Designed an interactive Coffee Order Dashboard in Microsoft Excel using Power Pivot, Pivot Tables, and advanced lookup functions to analyze sales trends, customer behavior, and product preferences.


Coffee Order Dashboard (Excel Project) 

Project Overview-
This project presents a Coffee Order Dashboard built using Microsoft Excel to analyze coffee sales data. The dashboard provides insights into customer behavior, product preferences, and sales trends.
Data cleaning and modeling were performed using Power Pivot, and advanced Excel functions were used to ensure accurate and efficient data handling.


🎯 Objectives
-Analyze coffee sales performance
-Identify top customers and countries
-Track sales trends over time
-Enable interactive filtering for better decision-making


🛠️ Tools & Technologies Used
1. Microsoft Excel
2. Power Pivot (Data Cleaning & Modeling)
3. Pivot Tables & Pivot Charts
4. Excel Functions:
   a. XLOOKUP
   b. INDEX + MATCH


📊 Dashboard Features

📈 Visualizations
1. Column Chart – Sales by Country
2. Column Chart – Top 5 Customers
3. Line Chart – Total Sales Over Time

Interactive Slicers
1. Roast Type (Medium, Dark, Light)
2. Loyalty Card (Yes / No)
3. Package Size (0.2kg, 0.5kg, 1kg, 2kg, etc.)


Key Insights
1. Identify high-performing countries
2. Recognize top 5 customers contributing to revenue
3. Analyze sales growth over time
3. Understand customer preferences based on roast type and package size


 Project Workflow
1. Data Collection (Excel / CSV)
2. Data Cleaning using Power Pivot
3. Data Transformation & formatting
4. Applying lookup functions (XLOOKUP, INDEX-MATCH)
5. Creating Pivot Tables
6. Building interactive dashboard with slicers


Key Features

1. Clean and simple one-page dashboard
2. Interactive slicers for dynamic analysis
3. Beginner-friendly project demonstrating core Excel skills
4. Clear visual representation of sales data


Business Value
This dashboard helps:

1. Understand customer purchasing behavior
2. Identify top-performing regions and customers
3. Optimize product offerings
4. Track overall sales performance


How to Use
